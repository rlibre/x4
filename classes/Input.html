    • Input

    Class Input

    Customizable input component supporting multiple types (text, number, date, etc.). Auto-generates CSS class: x4input.

    Example

    // Text input
    const nameInput = new Input({ type: "text", placeholder: "Enter name" });

    // Checkbox
    const agreeCheckbox = new Input({
      type: "checkbox",
      checked: true,
      change: (e) => console.log("Checked:", e.value)
    });
